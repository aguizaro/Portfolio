About Me

Hey, I'm Tony, a passionate software developer on a mission to craft engaging experiences and practical tools. My expertise lies in JavaScript, where I work with frameworks like Phaser for game development, leveraging build tools like Vite. I have built web applications using the Flask framework for python and SQLite for database management. I specialize in designing game systems and mechanics for both digital and analog games.

My experience in Python includes the development of various artificial intelligence tools. Noteworthy projects include a procedural level generator using genetic algorithms for a Super Mario World clone, an npc behavior model using behavior trees for a Planet Wars simulator, and a hierarchical task network planner for AI agents in Minecraft.

Beyond game development, I manage network servers for my home lab, running a suite of services including Pi-Hole, Apache web server, NAS for media storage, and an automatically synchronized Plex media server. I've handled diverse IT projects such as maintaining SMTP and POP servers for mail services, on-call 24 hour server maintenance , and assisting in a colocation data center facility. These projects have taught me: subnet design / IPV4 address , Unix/linux environment and tools, server troubleshooting, internet routing and security, network automation, and Wireless radio transmission. This understanding allowed me to effectively navigate and troubleshoot issues across the various network layers, contributing to my overall expertise in network architecture.

Portfolio

Personalized News App

This web application is a personalized site for reading news articles and blogs tailored specifically to topics I find interesting: technology, football, and video games. It is built using the Flask micro web framework for Python and relies on an SQLite database. The application includes a Python script that sends requests to the News API and writes the responses to the database. To retrieve the full article content (as News API truncates responses at 200 characters), the script performs HTTP requests to the article URL and parses the content using Beautiful Soup.

Chinatown 1974

This is a video game adaptation of the 1974 film 'Chinatown'. Chinatown is a 1974 American neo-noir mystery film directed by Roman Polanski from a screenplay by Robert Towne, starring Jack Nicholson and Faye Dunaway. The film was inspired by the California water wars, a series of disputes over southern California water at the beginning of the 20th century, by which Los Angeles interests secured water rights in the Owens Valley.

This game was made as a class project. It is written in JavaScript and uses the Phaser 3, a desktop and mobile HTML5 game framework. The game world is constructed using Tiled, a map editor that allows for the creation of intricate tilemaps, contributing to the immersive gaming experience.

stickerPAD

This sketch pad program is crafted with Typescript and developed using Vite. It offers a creative space for users to draw and express themselves. Enjoy the playful addition of emoji stickers to add a touch of fun to your sketches!

Rapid Delivery

I conceptualized and crafted this board game as part of a class project. In this game, players take turns collecting delivery tickets, fulfilling orders, and acquiring power-ups. The objective is to complete as many deliveries as possible by the end of the week, with each game round representing a different day of the week.

Offering a fast-paced experience, this board game is designed for 2-4 players. All game assets, from initial prototypes to final designs, were created using Adobe Illustrator.
